# Triage Labels

这些技能使用五个标准 triage 角色。此文件把标准角色映射到这个仓库 issue tracker 中实际使用的 label 字符串。

| mattpocock/skills 中的 label | 本仓库 tracker 中的 label | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要 maintainer 评估 |
| `needs-info` | `needs-info` | 等待 reporter 补充信息 |
| `ready-for-agent` | `ready-for-agent` | 信息完整，可以交给 AFK agent |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不会处理 |

当某个技能提到标准角色时，例如 “apply the AFK-ready triage label”，使用上表右侧对应的实际 label 字符串。

如果以后这个仓库采用了不同的 label 体系，直接修改上表右侧列即可。
